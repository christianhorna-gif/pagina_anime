## 🌟 Descripción

Una página web temática de anime desarrollada como proyecto de práctica utilizando HTML y CSS.

El sitio incluye:

✨ Diseño visual inspirado en anime  
✨ Sistema de inicio de sesión  
✨ Sistema de creación de cuenta  
✨ Galería de personajes  
✨ Interfaz moderna y atractiva
